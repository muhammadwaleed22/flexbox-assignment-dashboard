# Flexbox Assignment Dashboard

A clean and responsive Assignments Dashboard created using HTML and CSS Flexbox. This project displays multiple assignment cards in a structured dashboard layout.

## Preview

This project contains:
- Dashboard Header
- Assignment Cards
- Responsive Flexbox Layout
- Clean UI Design
- Box Shadow Effects
- Organized Card Structure

---

## Technologies Used

- HTML5
- CSS3
- Flexbox

## Features

- Responsive card layout using Flexbox
- Modern dashboard design

## Flexbox Concepts Used

This project uses important Flexbox properties such as:

```css
display: flex;
flex-wrap: wrap;
justify-content: center;
```

These properties help create a responsive and organized layout for the assignment cards.
